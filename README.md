# GitHub Pages 学术个人主页

这是崔小龙的学术个人主页静态版本，无需安装任何依赖。

## 更新个人信息

1. 用编辑器打开 `index.html`，可直接修改个人介绍、邮箱、研究方向和论文列表。
2. 如需更换头像，用新的照片覆盖 `profile.jpg` 即可，建议保持 4:5 比例。

## 发布到 username.github.io

1. 在 GitHub 新建公开仓库，名称必须是 `你的GitHub用户名.github.io`。
2. 把本目录中的所有文件上传到仓库根目录并提交。
3. 在仓库的 **Settings → Pages** 中选择 **Deploy from a branch**，分支选择 `main`，目录选择 `/ (root)`。
4. 等待几分钟后访问 `https://你的GitHub用户名.github.io/`。

如果使用普通仓库名，也可以在 Pages 设置中发布，网址会变成 `https://用户名.github.io/仓库名/`。
